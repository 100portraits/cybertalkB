Collaboration

Collaborative spaces are great tools for students to work together, brainstorm ideas, and share information. Similarly to a whiteboard, students can create mind maps, use sticky notes, and draw diagrams. It helps students to visualise and communicate their knowledge.

Tools we recommend:

Google Docs

Google Docs can be used to create and edit online word documents with peers. These files will be stored in the clouds, so they are easily shared and accessed to anyone with a computer or a phone.

Google Docs - Link

Miro

Miro is an online tool used to brainstorm, create concepts, and visualise ideas. With the use of sticky notes and indicator arrows, it has become a lot easier to share ideas that are difficult to put into words.

Miro - Link

Canva

Canva is a popular graphic design platform that allows users to create a wide range of visual content, including presentations, posters, social media graphics, and more. With Canva, users can choose from a vast library of templates, images, and design elements to create stunning visuals for various purposes. The platform makes it easy for users to customise their designs and create high-quality visuals without any prior design experience. Canva is a versatile tool that can be used by educators, students, and professionals to create engaging and visually appealing content for their audiences.

Canva - Link

Jamboard

Jamboard is a digital whiteboard developed by Google used for creating diagrams, brainstorming and visualising ideas. Users can create, edit, and share documents, images, and other types of media on the board, and work together.

Jamboard - Link
